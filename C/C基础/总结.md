# C语言的灵魂

- 函数
  - 功能封装的方式，程序功能由函数功能组合而成

- 指针
  - 扩展数据类型，每种数据类型都可以扩展指针类型
  - 指针拥有灵活访问内存的能力，是C语言底层开发的基础
